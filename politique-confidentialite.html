2. Quelles données collectons-nous ?

Nous collectons les catégories de données suivantes :

  • Données d'identification : nom, prénom, adresse e-mail (lors de la création de compte, quiz, achat, parrainage, newsletter)
  • Données de commande : produits achetés, montant, date d'achat
  • Données de paiement : nous ne stockons aucune donnée bancaire. Les paiements sont gérés exclusivement par PayPal.
  • Données de connexion et d'usage : adresse IP, type de navigateur, pages visitées, durée de visite, quiz réponses (anonymisées ou non selon le cas)
  • Données de fidélité / parrainage : nombre de points, code parrain, personnes parrainées (email uniquement si consentement)

4. Combien de temps conservons-nous vos données ?

  • Données clients / commandes → 5 ans après la dernière commande (prescription civile)
  • Données de fidélité → jusqu'à suppression du compte ou inactivité 3 ans
  • Newsletter → jusqu'au retrait du consentement
  • Logs techniques / IP → 12 à 26 mois maximum

5. Avec qui partageons-nous vos données ?

  • PayPal (paiement) – Politique PayPal
  • Hébergeur du site (données techniques uniquement)
  • Prestataires statistiques (Google Analytics, etc.) – données anonymisées ou pseudonymisées
  • Autorités judiciaires / administratives en cas d'obligation légale

Nous ne vendons jamais vos données personnelles à des tiers à des fins publicitaires.

6. Transferts hors Union Européenne

Certains destinataires (ex. Google, PayPal) peuvent transférer des données vers les États-Unis. Ces transferts sont encadrés par :

  • les Clauses Contractuelles Types de la Commission européenne, et/ou
  • le Data Privacy Framework (DPF) lorsque le prestataire y est certifié.
